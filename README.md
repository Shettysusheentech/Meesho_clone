# Meesho_clone
This repository contains a **fully responsive e-commerce website frontend** built using **HTML and CSS**. The project focuses on creating a modern shopping platform UI with a clean layout, smooth interactions, and mobile-first responsiveness.

###  Features

* Responsive **navigation bar** with search, profile, cart, dropdowns, and mega menu
* **Desktop, tablet, and mobile** optimized layouts using media queries
* Interactive **banner sections**, category navigation, and promotional strips
* Horizontally scrollable **gallery and brand sections**
* **Product listing page** with filters, ratings, pricing, and offers
* Mobile-friendly **filter modal** and toggle menu
* Structured and scalable **CSS architecture**
* Clean and modern **footer section** with links and social icons

###  Tech Stack

* **HTML5**
* **CSS3**
* Flexbox & Grid
* Media Queries for responsiveness

###  Purpose

This project is ideal for:

* Frontend UI practice
* Learning responsive web design
* E-commerce layout reference
* College or team-based web development projects

###  Responsive Design

* Desktop (≥1024px)
* Tablet (768px – 1024px)
* Mobile (≤767px)
